# Shubham Kumar — Portfolio Website

A clean, professional portfolio and resume website.

## 🚀 Deploying to Vercel

### Method 1: Via GitHub (Recommended)

1. Create a new GitHub repository (e.g. `shubham-portfolio`)
2. Upload these files to the repo:
   - `index.html`
   - `Shubham_Kumar_Resume.pdf`
   - `vercel.json`
   - `README.md`
3. Go to [vercel.com](https://vercel.com) → Sign up / Log in
4. Click **"Add New Project"**
5. Import your GitHub repository
6. Leave all settings as default → Click **"Deploy"**
7. Your site will be live at `https://your-project.vercel.app` ✅

### Method 2: Vercel CLI (Direct Upload)

```bash
npm install -g vercel
cd portfolio-folder
vercel
```
Follow the prompts — your site deploys in under 60 seconds.

### Method 3: Drag & Drop on Vercel

1. Go to [vercel.com/new](https://vercel.com/new)
2. Drag and drop this entire folder
3. Done ✅

- Edit `index.html` to update any content
- Replace `Shubham_Kumar_Resume.pdf` with a new PDF to update the resume
- The PDF is also embedded as base64 inside the HTML for guaranteed downloads
