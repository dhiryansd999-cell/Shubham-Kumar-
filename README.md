# Shubham Kumar — Portfolio Website

A clean, modern, fully responsive portfolio website built with HTML, CSS, and vanilla JavaScript.

## 🚀 Deploy to Vercel

### Step 1 — Upload to GitHub
1. Create a new repository on [github.com](https://github.com) (e.g., `shubham-portfolio`)
2. Upload these files to the repo:
   - `index.html`
   - `vercel.json`
   - `Shubham_Kumar_Resume.pdf` ← **your actual PDF resume**

### Step 2 — Deploy to Vercel
1. Go to [vercel.com](https://vercel.com) and sign up / log in with GitHub
2. Click **"Add New Project"**
3. Import your GitHub repo (`shubham-portfolio`)
4. Vercel auto-detects static HTML — just click **Deploy**
5. Your site will be live at `https://shubham-portfolio.vercel.app`

### Step 3 — Custom Domain (optional)
- In Vercel dashboard → Project → Settings → Domains
- Add your custom domain (e.g., `shubhamkumar.in`)

## 📁 File Structure
```
/
├── index.html                  ← Main portfolio page
├── vercel.json                 ← Vercel deployment config
└── Shubham_Kumar_Resume.pdf    ← Your resume (ADD THIS FILE)
```

## ⚠️ Important
- The **Download Resume** buttons link to `Shubham_Kumar_Resume.pdf`
- Make sure to add your **actual PDF resume** to the GitHub repo with that exact filename
- Without the PDF in the repo, the download button won't work on Vercel
